# bisect
bisect是python内置模块，用于有序序列的插入和查找。

bisect.bisect == bisect.bisect_right
相同的数值时，返回之后的idx

bisect.bisect_left 
相同的数值是，返回之前的idx

a= [1,4,5,7,7,9,23,56,78,100]
idx1 = bisect.bisect(a,7)
idx2 = bisect.bisect_left(a,7)
idx3 = bisect.bisect_right(a,7)
print(idx1,idx2,idx3)   # 5,3,5
