# 全排列的代码

第一种思路就是调用库函数
```
def permute(self, nums: List[int]) -> List[List[int]]:
    return list(itertools.permutations(nums))
```     

第二种思路就是
回溯遍历
```
class Solution:
    def permute(self, nums: List[int]) -> List[List[int]]:

        if len(nums) ==1:
            return [nums] 

        res = []
        def func(d,nums):
            if not nums:
                res.append(d)

            for i in range(len(nums)):
                func(d + [nums[i]],nums[:i] + nums[i+1:])
        func([],nums)

        return res 
```
